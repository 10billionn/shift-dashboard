# Shift Prototype

This prototype focuses on showing the intended workflow for a men's esthe shift generator.

## What this prototype shows

- Therapist input area
- Rule setting area
- One-click generation flow
- Weekly result table
- Guide area for first-time users

## Current input items

- Name
- Available days
- Preferred time
- Preferred area
- Note

## Current rule items

- Shift labels for early / late / off
- Required headcount for each day
- Simple assignment rule memo

## Structure for future extension

- `data.js`: sample data
- `app.js`: UI and screen flow
- `scheduler.js`: simple generation logic

## Future ideas

- Stronger shift optimization
- Room-based constraints
- Sales forecast
- CSV export
- Store-specific rule presets
